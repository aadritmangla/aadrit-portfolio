## 💎 Project Vision
This repository houses the official digital portfolio of **Aadrit Mangla**. Built with a "Luxury-First" philosophy, the site is engineered to serve as a high-performance casting tool for agencies and brands. The goal was to combine **editorial aesthetics** (Vogue-style minimalism) with **enterprise-grade technical performance**.

### 🎯 Core Objectives
- **Ultra-Low Latency:** Optimized for casting directors who need instant access to headshots.
- **SEO Dominance:** Geo-targeted for "Child Model New Delhi" to capture organic search traffic.
- **High Conversion:** Zero-friction booking flow via direct WhatsApp and Email integration.

## 🛠️ Tech Stack (The Engineering)
This project uses a modern, AI-native stack to ensure stability, speed, and scalability:

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend** | `React.js` + `TypeScript` | Type-safe, component-based architecture. |
| **Styling** | `Tailwind CSS` | Utility-first CSS for a responsive, luxury UI. |
| **Animation** | `Framer Motion` | High-end, cinematic reveal animations. |
| **Assets** | `Supabase Storage` | Cloud-optimized image delivery (WebP/AVIF). |
| **Build Tool** | `Vite` | Lightning-fast HMR and production bundling. |
| **Deployment** | `Vercel` | Edge-network hosting for global performance. |
| **SEO** | `JSON-LD` | Structured data for Google Knowledge Graph. |

---

## 📈 Performance & SEO Optimizations
To ensure a world-class user experience, the following optimizations were implemented:
- **LCP Optimization:** Critical images are set to `fetchPriority="high"` to ensure the hero shot loads in <1.5s.
- **Asset Compression:** All images are converted to **WebP** via lossy-less compression to reduce bandwidth by 70%.
- **Local SEO:** Implemented `geo.region` and `geo.placename` meta-tags to dominate New Delhi casting searches.
- **Responsive Design:** Fully optimized for mobile-first viewing (the primary device for casting agents).
